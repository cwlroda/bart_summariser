Basic Parameters

ⓘ Maximum leeway (in number of words) on the length of the summary.
ⓘ Minimum leeway (in number of words) on the length of the summary.

Advanced Parameters

ⓘ Number of beams for beam search. A value of 1 means no beam search. Larger beam search values mean analysing more variants of possible summaries, and will take longer to process.
ⓘ Ngrams refer to phrases of n words. if set to a value larger than 0, all phrases of words that are n words long can only occur once. For example, an article about the city of New York will only output the word "New York" once with a parameter value of 2.
ⓘ Penalty for repeating vocabulary in the generated summary. Larger values encourage the model to use different words from the original text. However, this may render gibberish if unconstrained. A value of 1.0 means there is no penalty.   
ⓘ Penalty based on length of generated summary. Larger values encourage the model to generate longer summaries, and vice versa. A value of 1.0 means there is no penalty.
